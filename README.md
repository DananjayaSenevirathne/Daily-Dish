# Daily-Dish

Daily-Dish is a full-stack food ordering website built with React.js, Node.js, Express.js, MongoDB, and Stripe. The project includes a customer-facing frontend, an admin panel, and a backend server for managing authentication, food items, orders, and payments.

## Features

- User registration and login authentication
- Browse food items by category
- Add and remove items from the shopping cart
- Place food orders easily
- Secure online payment integration with Stripe
- Admin panel to manage food items and orders
- Order status tracking and updates
- Full-stack architecture with frontend, backend, and database integration

## Tech Stack

### Frontend
- React.js
- CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Payment Gateway
- Stripe

## Project Modules

- **Frontend:** Customer interface for browsing food, managing cart, and placing orders
- **Admin Panel:** Dashboard for managing food items and updating order statuses
- **Backend:** REST API for authentication, food management, cart handling, orders, and payment processing
